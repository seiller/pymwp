Python $\geq$ 3.7

##### Installation 

Get the latest version from Python Package Index:

```console
pip install pymwp==0.3.0
```

<br/>On systems that do not default to Python 3 runtime, use:

```
python3 -m pip install pymwp==0.3.0
```

This completes the setup.
